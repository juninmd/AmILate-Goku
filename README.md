# AmILate
Plugin Cordova de exemplo feito em Java que exibe uma Toast nativa do Android 

***Exemplo de chamada***
```javascript
cordova.plugins.AmILate.coolMethod('GOKUUUUU', function (result) {
    console.log("sucesso", result);
}, function (err) {
    console.log("erro", err);
});
```
